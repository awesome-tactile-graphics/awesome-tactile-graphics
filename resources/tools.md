This document lists **software, libraries, embossers, and display systems** used in tactile graphics production and evaluation.  



## Libraries & Repositories

| Name | Description | Link | Notes |
|------|--------------|------|-------|
| **APH Tactile Graphics Image Library (TGIL)** | Curated collection of tactile-ready diagrams from the American Printing House for the Blind. | [tech.aph.org/tgil](https://imagelibrary.aph.org/portals/aphb/) | Useful for benchmarking AI-generated tactile graphics and for instructional design. |
| **BANA Tactile Graphics Guidelines (2010)** | Foundational North American tactile-graphics design standard. | [brailleauthority.org/tg/web-manual/index.html](http://www.brailleauthority.org/tg/web-manual/index.html) | Defines rules for spacing, labeling, and symbol conventions. |


## Authoring & Conversion Software

| Tool | Platform | Description | Link |
|------|-----------|--------------|------|
| **TactileView** | Windows / macOS | Full-feature tactile authoring tool with embosser integration and haptic preview. | [tactileview.com](https://www.tactileview.com/) |
| **Inkscape** | Cross-platform | Open-source vector editor with tactile export plug-ins and Braille font extensions. | [inkscape.org](https://inkscape.org/) |
| **Adobe Illustrator** | Windows / macOS | Industry-standard vector design software often used for tactile production templates. | [adobe.com/products/illustrator.html](https://www.adobe.com/products/illustrator.html) |
